

# t2-s1 Gestión de socios COPAL

## Variables de entorno
- **DB_URL**: Dirección y nombre de la base de datos
- **DB_USER**
- **PASSWORD**
- **SHOW_SQL**: false por defecto
- **SQL_INIT**: Ejecutar data.sql
    - always
    - never
- **DDL_MODE**:modo en que se crea la base de  datos
    - create
    - update
    - create-drop
